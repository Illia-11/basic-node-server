Етапи імпорту / запуску файлу у node:
  1. resolving - етап пошуку файлу / модулю
    Node буде намагатися імпортувати вашу річ за таким алгоритмом:
      1.1 Якщо ви намагаєтеся імпортувати модуль:
        1.1.1 Назва модулю шукається у вбудованих модулях node
        1.1.2 Назва модулю шукається у node_modules/
      1.2 Node вважає, що ви дали шлях до файлу:
        1.2.1 Шукати сам файл (якщо він з розширенням)
        1.2.2 Шукає файл з розширенням .js
        1.2.3 Шукає файл з розширенням .json
      1.3 Node вважає, що ви дали шлях до папки:
        1.3.1 Всередині папки буде шукатися файл "package.json" з властивістю
          "main"
        1.3.2 Всередині папки буде шукатися файл index.js
        1.3.3 Всередині папки буде шукатися файл index.json
      1.4 Node кидає помилку, що модуль не знайдено
  2. loading - знайдений модуль node завантажує собі в памʼять
  3. wrapping - код з файлу з минулого кроку загортається у функцію. При цьому
    йому додаються допоміжні змінні node
  4. evaluation - функцію з попереднього кроку запускають. (Для кожного
    конкретного файлу цей крок виконується тільки 1 раз)
  5. caching - результат 4 кроку запамʼятовується та повертається при натсупних
    імпортах
